# Таверна JRPG

## Швидке завантаження

Якщо ви хочете швидко забрати гру локально:

1. Відкрийте репозиторій у файловому менеджері.
2. Заархівуйте папку `myown` у ZIP, або в терміналі виконайте:

```bash
cd /workspace/myown
zip -r tavern-jrpg.zip .
```

Після цього файл `tavern-jrpg.zip` можна переслати або розпакувати на іншому ПК.

## Запуск локально

```bash
cd /workspace/myown
python -m http.server 8000
```

У браузері відкрийте `http://127.0.0.1:8000/index.html`.

## Керування

- Рух: **WASD** або **стрілки**
- Взаємодія: **E**
